# Roadmap

## v1 — First Release

### Must ship

- [ ] **Onboarding** — first-launch flow: pick JLPT target level and UI language
- [ ] **App icon** — custom icon at all required sizes (Android + iOS)
- [ ] **Home screen** — replace mock with real dashboard: today's due cards, progress overview, quick-access to active levels
- [ ] **Empty practice state** — when nothing is due, show a proper "all caught up" screen with next-review time instead of a 0/0 summary
- [ ] **Text-to-speech** — read Japanese words and sentences aloud on vocab tiles, kanji detail, and flashcards (`flutter_tts`)
- [ ] **Grammar screen** — at minimum N5 grammar points so the tab isn't an empty placeholder
- [ ] **Data sync** — sign in with Google / Apple, sync SRS progress so it survives reinstalls and transfers between devices
- [ ] **App name & bundle ID** — set before any store submission
- [ ] **Privacy policy** — required by App Store and Play Store

### Nice to have before v1

- [ ] **Kanji detail — example words** — already in DB, needs UI grouped by JLPT level

## Post-v1

- Onboarding refinement — level quiz instead of manual pick
- Grammar screen — deeper content beyond N5
- Streak / gamification
- Notifications — daily review reminders
